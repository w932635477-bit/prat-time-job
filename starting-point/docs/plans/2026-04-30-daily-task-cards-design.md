# Design: 7-Day Daily Task Cards (Phase 2)

Generated by /brainstorming on 2026-04-30
Status: APPROVED

## What Changes

Replace the 30-day content plan in CustomerAcquisitionSkill with a 7-day daily task card system. Each day has one concrete, low-barrier action that takes under 30 minutes.

## Backend

### customer_acquisition.py
- description: "7天逐日任务，每天一步，帮你接到第一个咨询"
- Step 3 text: "我帮你制定了7天行动计划。每天一个任务，30分钟内能完成。"
- generate_output: call DAILY_TASKS_TEMPLATE instead of CONTENT_WEEK_TEMPLATE
- Remove WEEK_THEMES constant

### prompts.py
- New DAILY_TASKS_TEMPLATE: receives platform, service_name, asset_map, market_signals, digital_literacy, time_commitment
- Output: 7 DailyTask objects

### models.py
- New DailyTask: day (int), task (str), platform (str), estimated_time (str), why (str), success_signal (str)
- New DailyTaskPlan: tasks (list[DailyTask]), platform (str)

## Frontend

### phases/customer-acquisition.js
- Replace 30-day accordion with 7-day task card list
- Each card: day label, task description, estimated time, why
- Checkbox per task, persisted in localStorage

### store.js
- New completedTasks field (array of booleans)
- New toggleTask(state, dayIndex) function

### phases/index.js
- Fix self_discovery steps from 8 to 11 (Phase 1遗漏)

## Not Changed
- Skill pipeline order
- First 3 step IDs and flow
